Infraestructura como Código (IaC)

Define y aprovisiona tu infraestructura de nube a través de código para lograr consistencia y automatización.

📜

AWS CloudFormation

Es el servicio principal de AWS para IaC. Te permite modelar tu infraestructura completa en archivos de texto (YAML o JSON). Al ejecutar estas "plantillas", CloudFormation aprovisiona y configura automáticamente todos los recursos de AWS descritos, como instancias EC2, bases de datos RDS y redes VPC.

  • Consistencia: Despliega el mismo entorno una y otra vez sin errores manuales.
  • Automatización: Integra el aprovisionamiento de infraestructura en tu pipeline de CI/CD.
  • Versionado: Trata tu infraestructura como código, guardando su historial en Git.

Monitoreo y Registros

Observar, medir y auditar son claves en DevOps. AWS ofrece servicios especializados para cada necesidad.

Enfoque del Servicio

CloudWatch vs. CloudTrail

Característica CloudWatch CloudTrail
¿Qué monitorea? El rendimiento de recursos y aplicaciones. La actividad en tu cuenta de AWS.
Pregunta que responde ¿Está alta la CPU de mi servidor? ¿Quién eliminó esa base de datos?
Datos principales Métricas, Logs, Eventos. Registros de llamadas a la API.
Caso de uso Alertar si una web va lenta; depurar errores de una app. Auditoría de seguridad; seguimiento de cambios.
